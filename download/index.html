Infinispan 8.0.x

“Infinite Darkness"
Unstable

8.0.0.Alpha1

Size-based eviction, Java 8 base, Indexless queries on indexed data

Zip Downloads:

All Contains all Infinispan components and demos
Minimal Contains just the core Infinispan libraries for embedding in your application
Server Contains the Infinispan server runtime for use as a remote data grid

Maven Coordinates

Add the following to the dependencies declared in your pom.xml file for the current stable version of infinispan-embedded 

...
<dependency>
  <groupId>org.infinispan</groupId>
  <artifactId>infinispan-embedded</artifactId>
  <version>7.2.1.Final</version>
</dependency>
...

or use the following for the current unstable version: 

...
<dependency>
  <groupId>org.infinispan</groupId>
  <artifactId>infinispan-embedded</artifactId>
  <version>8.0.0.Alpha1</version>
</dependency>

Maven Instructions

Maven Instructions

Infinispan artifacts are available on Maven Central.

If you wish to search for historical releases of Infinispan on Maven Central, you can search for the artifact you are looking for (e.g., infinispan-core ) on Maven Central's search interface.

If you are looking for a comprehensive list of all Infinispan artifacts and their respective versions, use the advanced search interface and look for group id org.infinispan
