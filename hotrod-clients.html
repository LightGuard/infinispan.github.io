Maven coordinates

Maven Coordinates

Add the following to the dependencies declared in your pom.xml file for the current stable version: 

...
<dependency>
  <groupId>org.infinispan</groupId>
  <artifactId>infinispan-client-hotrod</artifactId>
  <version>5.3.0.Final</version>
</dependency>
...

or use the following for the current unstable version: 

...
<dependency>
  <groupId>org.infinispan</groupId>
  <artifactId>infinispan-client-hotrod</artifactId>
  <version>6.0.0.Alpha3</version>
</dependency>
...

Hot Rod

Hot Rod is a wire protocol that Infinispan clients use to talk to a remote grid. It is a binary, platform-independent protocol that was developed in the open as a part of Infinispan.

The Hot Rod specification is available online, and several clients exist for this protocol.
