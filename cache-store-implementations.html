Cache Store Implementations

File Cache Store

Infinispan already ships with a couple of cache store implementations as a part of infinispan-core These are the FileCacheStore ( javadocs , config ) and the SingleFileCacheStore ( javadocs , config ). You don't need to download or install any additional libraries to make use of these cache stores.

Cloud Cache Store

This cache store implementation persists data to cloud storage services, such as Amazon's S3 service or Rackspace's CloudFiles service. Internally makes use of JClouds for communication with cloud storage services.

Maven coordinates

Maven Coordinates

Add the following to the dependencies declared in your pom.xml file for the current stable version: 

...
<dependency>
  <groupId>org.infinispan</groupId>
  <artifactId>infinispan-cachestore-cloud</artifactId>
  <version>5.3.0.Final</version>
</dependency>
...

or use the following for the current unstable version: 

...
<dependency>
  <groupId>org.infinispan</groupId>
  <artifactId>infinispan-cachestore-cloud</artifactId>
  <version>6.0.0.Alpha3</version>
</dependency>
...

Cache Stores

A cache store is how Infinispan optional persists data - either to disk or some other form of durable storage. A number of cache store implementations are available for you to use, and you can even implement your own custom cache store quite easily. A good way to do so is to look at the source code of an existing cache store and modifying it accordingly.
